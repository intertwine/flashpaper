The magic
How FlashPaper actually works
01
Ingest
We download the PDF (arXiv resolver included) and send it straight to Gemini using the native Files API. No text extraction loss — layout, tables, and figures are preserved.
02
Deep read
Gemini 3.5 Flash reads the entire paper with its massive context window, understanding every equation, ablation, and figure caption in one shot.
03
Design + code
A carefully engineered prompt turns the paper into a production-quality single-file website using Tailwind, Chart.js, Mermaid, KaTeX, and hand-crafted interaction patterns that actually work.
04
Live website
You receive a self-contained, shareable HTML file. Everything runs in the browser. No backend needed. Download it, host it anywhere, or keep exploring with the built-in chat.
