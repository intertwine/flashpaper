2016 • CVPR • HE, ZHANG, REN, SUN

Deep Residual Learning for Image Recognition

The ResNet paper that made 152-layer networks trainable.

THE PROBLEM
Deeper networks should be more powerful, but they started performing worse than shallow ones.

The degradation problem: as depth increases, training error rises even when the network has enough capacity.

THE KEY INVENTION

The Residual Block

Plain network (before ResNet)
Input → Conv → ReLU → Conv → ReLU → ...
Problem: gradients vanish as they flow backward through many layers.
Residual block (ResNet)
Desired mapping: H(x)
F(x) + x
The network only learns the residual F(x)
The skip connection lets gradients flow directly — training 8× deeper networks became possible.
